# 🗂️ Laurent Valquin — Portfolio

A creative frontend developer portfolio built with Next.js, featuring an artist's sketchbook concept with interactive project tabs.

🔗 [Live Demo](https://portfolio-laurent-plum.vercel.app)

---

## 🧱 Tech Stack

Next.js • TypeScript • TailwindCSS • Lottie • Vercel • Git/GitHub

---

## ✨ Features

- 🗂️ Artist sketchbook concept with clickable project tabs
- 🎬 Lottie animations
- 📄 Page transition animation
- 📱 Responsive design
- 🍔 Hamburger menu

---

## 🚀 Getting Started
```bash
npm install
npm run dev
```

---

## 📦 Deployment

Deployed on **Vercel** — automatic redeployment on every push to `main`.
